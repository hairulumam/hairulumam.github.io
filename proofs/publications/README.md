# Bukti Dokumen — Publikasi Ilmiah

Letakkan PDF publikasi ilmiah di folder ini.

## Konvensi Penamaan

Format: `TAHUN-deskripsi-pendek.pdf`
Aturan: huruf kecil semua, pakai dash `-` sebagai pemisah, tanpa spasi.

## Contoh

```
2026-sistem-masjid-tazkia.pdf
2026-ocr-google-cloud-vision.pdf
2026-queue-heterogeneous-arrival.pdf
2023-power-gompertz-makeham-journal.pdf
2023-power-gompertz-makeham-snmsa.pdf
```

## Cara Pasang Link

1. Letakkan PDF di sini
2. Buka `data/publications.json`
3. Ganti `"proof_url": null` jadi `"proof_url": "proofs/publications/NAMA-FILE.pdf"`

Setelah commit & push ke GitHub, link akan otomatis aktif di website.
