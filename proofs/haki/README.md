# Bukti Dokumen — HAKI / Karya Cipta

Letakkan sertifikat HAKI / Karya Cipta dari Kementerian Hukum RI di folder ini.

## Konvensi Penamaan

Format: `TAHUN-jenis-deskripsi-NOMOR.pdf`

## Contoh

```
2026-hak-cipta-sistem-masjid-001223947.pdf
```

## Cara Pasang Link

1. Letakkan PDF di sini
2. Buka `data/haki.json`
3. Ganti `"proof_url": null` jadi `"proof_url": "proofs/haki/NAMA-FILE.pdf"`
