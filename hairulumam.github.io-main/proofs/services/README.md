# Bukti Dokumen — Pengabdian Masyarakat

Letakkan dokumentasi kegiatan pengabdian, foto kegiatan, laporan, atau MOU di folder ini.

## Konvensi Penamaan

Format: `TAHUN-deskripsi-pendek.pdf`

## Contoh

```
2026-sistem-masjid-dokumentasi.pdf
2025-laporan-youtube-edukasi.pdf
2025-komunitas-telegram-statistik.pdf
```

## Cara Pasang Link

1. Letakkan PDF di sini
2. Buka `data/services.json`
3. Pada card yang relevan, tambahkan field: `"proof_url": "proofs/services/NAMA-FILE.pdf"`
