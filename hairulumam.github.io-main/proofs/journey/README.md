# Bukti Dokumen — Riwayat Karier & Pendidikan

Letakkan SK pengangkatan, ijazah, transkrip, atau dokumen resmi lain di folder ini.

## Konvensi Penamaan

Format: `TAHUN-jenis-institusi.pdf`

## Contoh

```
2023-sk-dosen-stmik-tazkia.pdf
2021-ijazah-s2-itb.pdf
2017-ijazah-s1-ugm.pdf
2021-transkrip-s2-itb.pdf
2017-transkrip-s1-ugm.pdf
2021-chairman-balada-madura.pdf
2018-cpo-cerebrum.pdf
```

## Cara Pasang Link

1. Letakkan PDF di sini
2. Buka `data/journey.json`
3. Ganti `"proof_url": null` jadi `"proof_url": "proofs/journey/NAMA-FILE.pdf"`

## Catatan Privasi

- **JANGAN upload dokumen yang mengandung data pribadi sensitif** seperti NIK, NIDN lengkap, atau tanda tangan basah.
- Untuk dokumen sensitif, lebih aman pakai Google Drive private + share link "Restricted" (request access).
